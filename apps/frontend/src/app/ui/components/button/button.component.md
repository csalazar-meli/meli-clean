# Button Component

Render button with design of full-moon.

---

## Usage

`html`

```html
<full-moon-button #myBtn [theme]="myBtn.types.SECONDARY">
  Buscar
</full-moon-button>
```

## Properties

| theme       |                                                       |
| ----------- | ----------------------------------------------------- |
| Description | Define theme of button view [ButtonType](#buttonType) |
| Attribute   | theme                                                 |
| Type        | `string \| ButtonType`                                |
| Default     | `ButtonType.PRIMARY`                                  |
